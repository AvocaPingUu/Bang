# client

Electron + React + TypeScript + Vite Frontend.

- `src/components/` — React UI-Komponenten (Spieltisch, Karten, HUD)
- `src/animations/` — Framer Motion Animationen
- `src/assets/` — SVG Karten, Icons, Hintergründe
- `src/game/` — Client-seitige Spielzustand-Darstellung (kein Spiellogik)
- `electron/` — Electron Main-Prozess, Preload-Skripte
